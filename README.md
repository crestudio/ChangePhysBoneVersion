## 개요 및 작동방식

VRChat PhysBone 컴포넌트의 PhysBone 버전 일괄 변경 스크립트

현재 활성화 된 씬(Scene)에서 숨겨진 오브젝트를 포함하여 **모든 피직스본 컴포넌트에 적용**

___

## 사용방법

1. 사용 중인 Unity 프로젝트에 해당 패키지를 임포트합니다
1. **Unity 메뉴의 Tools → VRSuya → PhysBone 메뉴**에 들어갑니다
1. 원하는 기능을 선택합니다
